System Architecture Overview

Backend Stack

  • Framework: Laravel 11.x
  • Real-time: Laravel Reverb (WebSocket)
  • Queue: Redis / Horizon
  • Database: MySQL + Scout (database)

Frontend Stack

  • Framework: Vue 3 (Composition API)
  • Build Tool: Vite
  • State: Pinia
  • Styling: TailwindCSS + Rekaui
