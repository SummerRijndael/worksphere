System Overview

The Coresync V2 Calendar System is a robust scheduling tool that integrates directly with Teams and supports Google Calendar synchronization. It offers drag-and-drop event management, ICS exports, and multi-user event sharing.

API Endpoint Map (Team Context)

Method Endpoint Description
GET /api/teams/{team}/calendar List team events
POST /api/teams/{team}/events Create new event
PUT /api/teams/{team}/events/{event} Update event
DELETE /api/teams/{team}/events/{event} Delete event
GET /api/teams/{team}/events/{event}/ics Download .ics file

Database Schema

events

  • id, public_id (ulid)
  • title, description
  • start_at, end_at (timestamp)
  • is_all_day (boolean)
  • location (string)
  • color (hex)

event_attendees

  • event_id (fk)
  • user_id (fk)
  • status (pending/accepted/declined)

calendar_shares

  • id
  • user_id (owner)
  • shared_with_user_id (recipient)
  • permissions (read/write)

Key Features

  • Drag & Drop Intuitively reschedule events by dragging them on the view.
  • Google Sync Two-way synchronization with personal Google Calendars.
  • ICS Export Download events as standard .ics files for external use.
