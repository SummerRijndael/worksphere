System Overview

The Coresync V2 Email Client is a full-featured IMAP/SMTP unified inbox with support for multiple email providers including Gmail (OAuth), Outlook (OAuth), and custom IMAP servers. It uses a provider adapter pattern to handle provider-specific behaviors.

Email Provider Adapter Architecture

The adapter pattern abstracts provider-specific logic (authentication, folder naming, message fetching) behind a common interface, allowing jobs to work uniformly with any provider.

Backend Architecture

Controller Layer

Main CRUD for emails. Handles listing, searching, filtering (Labels/Folders/Dates), and viewing single emails with pagination.

Queue Jobs

Sanitization

Cleans HTML bodies, blocks external trackers, and preserves inline CID images.

Core Features

  • Multi-Provider Support Gmail OAuth, Outlook OAuth, and Custom IMAP with adapter pattern.
  • Infinite Scroll Paginated email list with IntersectionObserver for seamless loading.
  • Organization Custom Folders, Color-coded Labels, and bulk actions.
  • Rich Composer Tiptap editor with attachments, signatures, and formatting.
  • Background Sync Queue-based email synchronization with chunked processing.
  • Data Export Export individual emails as standard .eml files.
